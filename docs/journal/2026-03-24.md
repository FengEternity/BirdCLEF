## 遇到的问题

1. **Kaggle 认证方式变更**：新版 Kaggle 使用 `KGAT_` 前缀的 token + `KAGGLE_API_TOKEN` 环境变量，而不是传统的 `kaggle.json` 里 username+key 的格式
2. **Arch/Manjaro PEP 668**：系统级 Python 环境不允许直接 `pip install`，需要使用 `pipx` 安装全局 CLI 工具
3. **Shell 工具限制**：Cursor 内的 Shell 工具无法捕获命令输出，需要在外部终端运行命令
